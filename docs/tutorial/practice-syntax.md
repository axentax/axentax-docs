# 機能的な奏法

## マップ機能

### マップ機能の基本

`:map()`を使用すると、1つまたは複数のnoteに対してループ処理を指示することができます。

```
@@ {
  |||2||2:map(*4) // 4回リピート
  |||2||2:map(0..3) // 音階を変更しつつ4回リピート（マイナス指定も可能）
}
```
上記の「音階を変更しつつ4回リピート」の場合、スケールに準じて音階が移動します。

:::info スケール指定方法は後述します。
:::

複数音に対してまとめてブロックに指定することもできます。

```
@@ {
  {
    |||2||2
    0|
  }:1/16:map(0..11)
}
```
